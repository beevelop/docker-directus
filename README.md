# directus

----
### Pull from Docker Hub
```
docker pull beevelop/directus:latest
```

### Build from GitHub
```
docker build -t beevelop/directus github.com/beevelop/docker-directus
```

### Run image
```
docker run -it beevelop/directus bash
```

### Use as base image
```Dockerfile
FROM beevelop/directus:latest
```
